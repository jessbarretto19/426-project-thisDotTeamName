## Domain Description
Our project simulates a community theater costume and prop lending system that allows schools, nonprofit theater groups, and local performance organizations to browse, reserve, and share costumes, props, and set pieces. As more organizations search the catalog, make reservations, and update item availability at the same time, a single server could become overloaded with requests. The system must remain reliable so that inventory and reservations stay accurate and organizations do not accidentally reserve the same items. This project supports the goals of Computing for the Common Good by helping community arts groups, such as the UMass Theatre Guild and Amherst Repertory Company (two student run theatre companies on campus) share resources instead of purchasing new ones, making theater more affordable and accessible. A reliable system benefits volunteers, performers, educators, and local arts organizations, while failures could lead to scheduling conflicts, duplicate reservations, and canceled performances.
